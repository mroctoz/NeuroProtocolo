# NEUROPROTOCOLO
> Engenharia Neural, Inteligência Emocional e Alta Performance.

O NeuroProtocolo é uma plataforma de desenvolvimento humano baseada em neurociência comportamental e neuroplasticidade. Diferente de listas de tarefas comuns, este sistema opera como um algoritmo de reconfiguração neural de 28 dias, projetado para sair do piloto automático, regular a química cerebral e otimizar a inteligência social.

---

## SOBRE O PROJETO

Este projeto é uma Progressive Web Application (PWA) que roda inteiramente no navegador (Client-Side), sem necessidade de banco de dados externo ou instalação complexa.

O objetivo é traduzir conceitos densos de neuroanatomia e psicologia (baseados nas obras de Daniel Goleman e Oliveira & Santos) em ações práticas diárias, utilizando gamificação, leitura imersiva e engenharia de áudio para maximizar a retenção.

### Principais Funcionalidades

* Jornada de 28 Dias: Um currículo progressivo dividido em 4 módulos neurocientíficos.
* Leitor Imersivo: Interface de leitura focada (Deep Reading) para absorção de conteúdo denso.
* Neuro-Acústica: Player de áudio integrado com frequências binaurais e sons de foco que persistem entre as navegações.
* Gamificação Neural: Sistema de XP, Níveis e Estatísticas baseadas em 4 pilares (Autoconsciência, Resiliência, Social, Foco).
* Offline-First: Todos os dados (progresso, diários) são salvos no localStorage do navegador. Privacidade total.
* Design Neuro-Tech: Interface "Glassmorphism" escura, projetada para reduzir o cansaço visual e aumentar o foco.

## A METODOLOGIA (OS 4 MÓDULOS)

O protocolo segue uma curva de carga cognitiva progressiva:

### Módulo 1: Fundação Neural (Dias 1-7)
Foco: Anatomia e Autoconsciência.
* Entendimento do custo energético do cérebro.
* Diferenciação entre Sistema Límbico (rápido/emocional) e Neocórtex (lento/racional).
* Identificação do "Sequestro da Amígdala".

### Módulo 2: Engenharia Química (Dias 8-14)
Foco: Neurotransmissores e Regulação.
* Modulação da Dopamina (Desejo vs. Prazer).
* Gestão da Habênula Lateral (O centro da frustração).
* Controle do Eixo HPA (Cortisol e Estresse).

### Módulo 3: O Cérebro Social (Dias 15-21)
Foco: Conexão e Empatia.
* Ativação dos Neurônios Espelho.
* A biologia da Inveja (Córtex Cingulado Anterior).
* A química da confiança (Ocitocina) e comunicação não-defensiva.

### Módulo 4: Transcendência (Dias 22-28)
Foco: Alta Performance e Espiritualidade.
* Neuroteologia e o Lobo Parietal.
* A ilusão do Livre Arbítrio e o poder do Veto Consciente.
* Superando a procrastinação e a Rede de Modo Padrão (DMN).

## TECNOLOGIAS UTILIZADAS

O projeto foi construído com a filosofia "Vanilla Power" para garantir performance máxima e zero dependências.

* HTML5 Semântico: Estrutura modular.
* CSS3 Moderno: Variáveis (Custom Properties), Flexbox, Grid, Animações Keyframes e Glassmorphism.
* JavaScript (ES6+):
    * Classes e Módulos.
    * Async/Await para carregamento de dados JSON.
    * Manipulação de DOM reativa.
    * localStorage API para persistência de dados.
* JSON: Arquitetura de dados para armazenar livretos e lógica de desafios.

## COMO RODAR O PROJETO

Não é necessário instalar Node.js, NPM ou servidores complexos.

1. Clone o repositório ou baixe os arquivos.
2. Abra o arquivo principal: Navegue até a pasta e dê um duplo clique em `index.html`.

NOTA: Para que o sistema de áudio e o carregamento de JSON funcionem perfeitamente em alguns navegadores (como Chrome) que bloqueiam requisições de arquivos locais por segurança (CORS), recomenda-se usar uma extensão como "Live Server" no VS Code ou rodar um servidor local simples (ex: python -m http.server).

---

## MODO DE DESENVOLVEDOR (GOD MODE)

Para fins de teste e validação de conteúdo, o sistema possui um "God Mode" oculto.

1. Abra o Dashboard.
2. Pressione simultaneamente: Shift + X + Z.
3. Confirme o alerta.
4. Resultado: Todos os 28 dias serão desbloqueados, seu nível subirá e XP será adicionado, permitindo navegar por todo o conteúdo imediatamente.
